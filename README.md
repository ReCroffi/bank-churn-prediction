# Costumer Churn Preditiction 


## Problema de Negócio:
A taxa de rotatividade de clientes (ou Churn) é uma das métricas que indicam quanto de receita ou clientes em um período de tempo. É um dos KPIs mais importantes para empresas que trabalham com modelo de assinaturas.

Nosso negócio sob investigação é um banco (ABC Multinacional Bank).

## Objetivo:
O objetivo deste projeto é prever a evasão de clientes (churn) e identificar os principais fatores que influenciam a saída, permitindo ações de retenção.

## Impacto:
Cada cliente que deixa o negócio significa perda de receita.
